#### 重建二叉树
题目：输入某儿二叉树的前序和中序遍历结果，重建出该二叉树并输出它的头结点。
我们知道前序遍历序列中，第一个数字是根节点的值。所以在中序遍历序列中，我们可以知道那个值是根节点，左子树的值位于根节点的值的左边，右子树的值位于根节点的值的右边。
问题的解决入口既是在前序遍历和中序遍历两个序列中，分别找到左右子树对应的子序列。然后用同样的方法去构建左右子树，自然是用递归解决了。
用于测试的两个遍历序列如下：
int PreOrder[] = { 1, 2, 4, 7, 3, 5, 6, 8 };
int InOrder[] = { 4, 7, 2, 1, 5, 3, 8, 6 };
