#### 快速排序思路
快速排序的思想就是分治法，PartSort函数可以把区间划分成三个部分，div左边的数都比div小，右边的数都比div大，并返回数组下标div。也就是说div位置的值已经放在了合适的位置，只要div的左边和右边都有序了，那么整个数组就有序了。解决这两个子问题就可以了，子问题如果区间比较大同样用快速排序的PartSort函数进行切割问题，反之，直接用插入排序同样可以解决子问题。
1. 如果区间大小为1或交叉就return（递归结束条件）；
2. 区间大小小于等于某个值就进行插入排序；
3. 否则，调用PartSort函数把这个区间划分成三个部分：[left, div - 1]，div, [div + 1, right]。div左边的数都比div小，右边的数都比div大。并返回数组下标div；然后递归快速排序[left, div - 1]和[div + 1, right]

快速排序有两个优化的地方：
小区间优化（插入排序使小区间有序）；
三数取中算法避免这样情况：在数组已经有序的情况下，每次划分将得到最坏的结果（递归的深度变深）
